# routes

Definicion de rutas y su conexion con controladores.
Cada archivo agrupa rutas de un modulo.

## Mapa actual de rutas
- `GET /` -> ruta raiz
- `GET /api/health` -> health check
- `POST /api/auth/register` -> registro de tutor
- `POST /api/auth/login` -> login de tutor
- `GET /api/protected/me` -> smoke test JWT
